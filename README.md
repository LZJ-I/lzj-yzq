# lzj-yzq

LZJ 与 YZQ 的恋爱纪念日计时器，记录从在一起那天起走过的每一天。

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/LZJ-I/lzj-yzq)

## 配置

编辑根目录 `config.js` 即可修改姓名、起始时间、座右铭，无需改 HTML。

## 本地预览

```sh
./start.sh
```

浏览器打开 http://127.0.0.1:8000

## 部署

### 一键部署（推荐）

点击上方 **Deploy with Vercel** 按钮，用 GitHub 账号授权后按提示完成即可。

### 手动部署到 Vercel

1. 将本仓库推送到 GitHub
2. 登录 [Vercel](https://vercel.com)，Import 该仓库
3. 无需构建命令，Framework Preset 选 **Other**，直接 Deploy

### 其他静态托管

将仓库内所有文件（含 `index.html`、`config.js`、`assets/`）上传至任意静态站点即可。
